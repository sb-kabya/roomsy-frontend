# Roomsy Frontend

A modern hotel booking web application built with **ReactJS** and **Tailwind CSS**.

🌐 **Live Site:** https://roomsy-frontend.vercel.app  
🔗 **Backend API:** https://roomsy-iota.vercel.app  


---

## 🚀 Tech Stack

| Technology | Purpose |
|---|---|
| ReactJS | Frontend framework |
| Tailwind CSS | Styling |
| React Hot Toast | Notifications |
| React Icons | Icon library |

---

## 📄 Pages

| Page | URL | Description |
|---|---|---|
| Home | `/` | Landing page with 7 sections |
| Hotels | `/hotels` | Browse and search all hotels |
| Hotel Detail | `/hotels/:id` | Hotel info, rooms, reviews |
| Booking | `/booking/:roomId` | 3-step booking process |
| Payment | `/payment` | Wallet top-up with SSLCommerz |
| My Bookings | `/my-bookings` | View and cancel bookings |
| Profile | `/profile` | User profile and wallet |
| Login | `/login` | Sign in |
| Register | `/register` | Create account |
| About | `/about` | About Roomsy |
| Contact | `/contact` | Contact form and FAQ |
| Dashboard | `/dashboard` | Admin analytics (admin only) |

---


## 🔐 Features

- JWT authentication with auto token refresh
- Email activation on registration
- Role-based access (Admin / User)
- Hotel search and filtering by city, name, star rating
- 3-step booking with guest info collection
- Wallet system with deposit
- Booking cancellation with auto refund
- Hotel reviews and star ratings
- Admin dashboard with analytics
- SSLCommerz payment integration
